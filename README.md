Repository: nex_frontend

Description:This repository contains the frontend code for a Next.js-based web project. 
It is structured to handle a modern web application with a clear separation of components and configurations.

Structure:

/public/ – Contains static files such as images, icons, and other assets that are not processed directly by Next.js.

/src/ – The main directory for source code, typically containing components, pages, and styling.

/src/components/ – Reusable UI components.

/src/pages/ – Next.js page structure, where each file automatically becomes a route in the application.

/src/styles/ – Styling files, possibly using CSS, SCSS, or Tailwind CSS.

Configuration Files:

.editorconfig – Configures code formatting across different editors.

.eslintrc.js – ESLint configuration file to enforce coding standards.

.gitignore – Defines files to be ignored by Git.

next.config.mjs – Next.js configuration file.

package.json – Contains dependencies and scripts for running and building the project.

tsconfig.json – TypeScript configuration.

Technologies Used:

Next.js – A React-based framework for server-side rendering and static pages.

TypeScript – Adds strong typing for more robust code.

Tailwind CSS (if used) – A utility-first CSS framework for styling.

ESLint & Prettier – For code standards and formatting.
